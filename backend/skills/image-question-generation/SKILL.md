# Image Question Generation for English Learners

## Overview

This skill guides you in creating effective English practice questions from images. Questions should push the student to use varied vocabulary, construct complete sentences, and think critically — not just label objects.

## Question Taxonomy

Generate exactly 5 questions using this mix:

### Type 1: Description Questions (2 questions)

Ask the student to describe what they observe. These build basic descriptive vocabulary and sentence construction.

**Good examples:**
- "What is happening in the center of the image?"
- "Describe the setting — where do you think this scene takes place?"
- "What details do you notice in the background?"

**Bad examples (too simple):**
- "What color is the car?" (single-word answer)
- "Is there a dog?" (yes/no)

**Tip:** Frame questions so the student must write at least one full sentence. Use "describe", "what is happening", or "tell me about" phrasing.

### Type 2: Vocabulary Question (1 question)

Ask the student to choose or produce specific words. These target vocabulary precision and range.

**Good examples:**
- "What adjective best describes the mood or atmosphere of this scene?"
- "How would you describe the texture of the surface in the foreground?"
- "What verb captures the action the person on the left is performing?"

**Bad examples:**
- "What is the word for this?" (pointing at nothing specific)
- "Name 10 objects." (list, not a sentence)

**Tip:** Target abstract concepts (mood, texture, relationship) rather than just naming concrete objects.

### Type 3: Reasoning Questions (2 questions)

Ask the student to infer, speculate, or explain. These develop higher-order language skills — conditionals, cause/effect, opinion language.

**Good examples:**
- "Why do you think the person in the image is smiling?"
- "What do you think happened just before this moment?"
- "If you were in this scene, what would you do and why?"
- "What can you infer about the relationship between the two people?"

**Bad examples:**
- "Do you like this image?" (opinion without language depth)

**Tip:** Use "why", "what do you think", "what might", "if you were" to elicit complex sentence structures.

## Difficulty Calibration

Assume an **intermediate ESL learner** (B1-B2 CEFR level):
- Can understand main points of familiar topics
- Can describe experiences and events
- Can briefly explain opinions and plans
- May struggle with complex grammar (conditionals, passive voice, relative clauses)

Calibrate questions so:
- Description questions are accessible (B1)
- Vocabulary question is moderately challenging (B1-B2)
- Reasoning questions push into B2 territory

## Sequencing

Order questions from easiest to hardest:
1. Start with a broad description question (warm-up)
2. Follow with a focused description question
3. Vocabulary question in the middle
4. First reasoning question
5. End with the most open-ended reasoning question

This builds confidence before tackling harder questions.
